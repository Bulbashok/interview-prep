## Описание моей работы в проекте

### Процесс и технологии

Разрарабтывал вместе с командой приложение для прохождения собеседований.
Использовал стек: React + TypeScript + Vite, Material-UI, Firebase Firestore, SCSS, react-i18next, react-hot-toast.

### Что было сложного

1. **React Learning Curve**: Первый опыт с React, изучал на практике
2. **Material-UI**: Кастомизация компонентов и система тем
3. **Firebase Integration**: Настройка структуры данных и асинхронные операции
4. **i18n + Фильтры**: Логика переводов при динамической фильтрации
5. **Архитектура Library**: Продумывание фильтров и состояния

### Что сделал с нуля

- Landing Page полностью
- Toast System архитектуру
- Library Page с фильтрацией
- RootLayout компонент
- Структуру данных на Firebase
